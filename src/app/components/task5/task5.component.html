assignment Task 5 - UI/UX assignment


Question:
Please go over the following design,
and provide your insights regarding the interface (no more than 4-5 points).
http://nihongo-e-na.com/archives/006/201206/large-4fe806d39a5fc.png



Answer:

  1. Internationalization - There is no option (in any other language) to change
    the page's language.

  2. Accessibility - This page is not accessible to people with disabilities.
    There is no special menu for changing the website's font size, brightness,
    sound (reading) options, etc.

  3. GUI - Most of the info/divs/boxes are very scattered, crowded, messy and not
    ordered by anything, like subjects in headlines (horizontally is best).

  4. Usability - There is too much info in the header and upper menus.
    They should contain less categories and more opened sub-menus.
    This page contains too much info, divs, pictures and subjects.
    It is very confusing and uncomfortable for the user.

  5. There is no separation between main content, sub content and commercials.
